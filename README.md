# NotPad

The app is a notpad to help users draft notes and save to a server and retrieve when needed. 

## Getting Started

## Setup

To get started, you should first get this file in your local machine by cloning or forking this project or typing in your terminal

```
  git clone https://github.com/elisha2kyakpo1/notPad.git

  cd notPad

  Open index.html which is in dist folder
```
